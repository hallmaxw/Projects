Created by Maxwell Hall

Developed on Mac OS X 10.10 (Linux) with Java version 1.8.0_11

HOW TO COMPILE:
1) Put HMM.java and Solution.java in the same directory
2) Execute the following command: javac Solution.java

HOW TO RUN:
1) Move to the directory containing the Java Byte code.
2) Execute the following command: java Solution <model file> <observations file>
