PLATFORM:
    This was developed on Mac OS X version 10.10

HOW TO COMPILE:
    1) Have Perceptron.java and Solution.java in the same directory.
    2) Open the command line in the same directory as the source files.
    3) Execute this command on the command line: javac Solution.java

HOW TO RUN:
    1) Open the command line in the same directory as the Byte code files.
    2) Run this command with your parameters: java Solution <training file> <test file> <learning rate> <iterations>
