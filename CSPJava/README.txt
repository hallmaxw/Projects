Developed on Mac OS X 10.10 (Linux)

HOW TO COMPILE:
1) Place Constraint.java, ConstraintGraph.java, Variable.java, and Solution.java in the same directory.
2) Execute the following command line argument: javac Solution.java

HOW TO RUN:
1) Navigate to the directory containing the java Byte code.
2) Execute the following command: java Solution <variable file> <constraint file> [none|fc]
